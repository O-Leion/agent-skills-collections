# Connect SSH

`.env` ファイルのSSH接続情報を使用して、リモートLinux/Ubuntuマシンへの SSH 接続を確立するAIエージェント向けSkillです。

## 概要

ローカルのVSCode系エディタで開発中に、リモートマシンへSSH接続したい場面で使用します。
AIエージェントが `.env` から接続情報を読み取り、`ssh` コマンドを組み立てて実行します。

```
ローカル（VSCode）                    リモート（Ubuntu/Linux）
┌──────────────────┐                ┌──────────────────┐
│ .env から読み取り  │  ── ssh ──→  │ SSH接続確立       │
│ エージェントが実行 │               │ パスワードは      │
│                   │               │ ユーザーが手入力   │
└──────────────────┘                └──────────────────┘
```

## 対象エージェント

- Antigravity
- RooCode
- GitHub Copilot

## セットアップ

### 1. `.env` ファイルの作成

`assets/.env.example` をプロジェクトルートにコピーして、実際の接続情報を記入してください。

```bash
cp assets/.env.example .env
```

```env
SSH_HOST=192.168.1.100    # リモートマシンのIPアドレス
SSH_USER=ubuntu           # SSHユーザー名
SSH_PORT=22               # ポート番号（デフォルト: 22）
```

> ⚠️ `.env` には接続情報が含まれるため、`.gitignore` に追加してGit管理対象外にしてください。

### 2. Windows環境の場合

WSL（Windows Subsystem for Linux）の起動が必要です。詳細は `SKILL.md` を参照。
