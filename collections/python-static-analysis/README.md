# Python Static Analysis

flake8 を使用して Python コードの静的解析を行い、指摘に基づいてコードを修正するAIエージェント向けSkillです。

## 概要

Python開発中に flake8 で静的解析を実行し、コード品質の問題を検出・修正します。
エージェントが flake8 を実行 → 結果を提示 → ユーザー確認後にコードを修正する流れです。

```
エージェント: flake8 を実行して指摘を取得
    ↓
エージェント: 結果をユーザーに提示
    ↓
ユーザー:    修正の承認
    ↓
エージェント: コードを修正（シンプル→自動、複雑→相談）
    ↓
エージェント: 再チェックで指摘ゼロを確認
```

## 対象エージェント

- Antigravity
- RooCode
- GitHub Copilot

## 前提条件

| 項目   | 要件                                     |
| ------ | ---------------------------------------- |
| Python | 3.10+                                    |
| flake8 | インストール済み（`pip install flake8`） |

## 主な機能

- flake8 による静的解析の実行（`E501` は除外）
- 指摘内容の分類（自動修正 / ユーザー相談）
- 修正後の再チェックによる品質確認
